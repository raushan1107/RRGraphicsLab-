Demo 19 · Module 13
UBO + Descriptor Sets — Replacing glUniform*()
Builds the complete Vulkan descriptor system: VkDescriptorSetLayout (binding 0 = uniform buffer), VkDescriptorPool, one VkDescriptorSet per swapchain image. Each frame: updates the UBO (MVP matrices) via persistent mapped memory, binds the descriptor set, draws an animated rotating orange triangle.
VkDescriptorSetLayoutVkDescriptorPoolUBOvkUpdateDescriptorSets
🎯 What you'll see: Orange triangle rotating at 45°/sec driven by MVP matrix in UBO. Terminal prints frame count. Validation: 0 errors.
⚡ Open Demo 📖 Notes ~40 min
Demo 20 · Module 13
Vulkan Compute Shader — GPU Parallel Image Processing
Creates a Vulkan compute pipeline (no graphics pipeline, no window, no swapchain). Dispatches a compute shader across a 256×256 image: each of 65,536 GPU threads inverts one pixel. Storage buffers (SSBO) hold input and output. Pipeline barrier ensures GPU writes complete before CPU reads. Terminal reports timing and throughput.
VkComputePipelineSSBOvkCmdDispatchPipeline barriers
🎯 What you'll see: Terminal only. Input/output pixel values verified, dispatch time in µs, throughput in pixels/second. No window required.
