Lab 1 Day 2 · Module 4

Place a 3D Vessel Shape in World Space

Build a recognisable 3D shape using a cube mesh. Apply the full MVP pipeline so it sits in a 3D world with a proper camera and perspective projection. This is the geometric foundation the lighting and texture will be applied on top of in Labs 2 and 3.

📋 Operational Requirements

The display system must:

  • →Render a 3D box-shaped vessel silhouette (a cube is your approximation)
  • →Apply a model matrix so it rotates slowly on the Y-axis (continuous rotation for visual identification)
  • →Place the camera at a fixed briefing angle: slightly above and in front of the vessel
  • →Use perspective projection (45° FOV) to give realistic depth appearance
  • →Enable depth testing so near faces correctly cover far faces
1 Create your project and cube vertex data ›

Create C:\Labs\Day2_Lab1_VesselShape\src\main.cpp. Copy your Day 1 Lab CMakeLists.txt (rename project to day2lab1). Your cube needs 36 vertices (6 faces × 2 triangles × 3 vertices). Each vertex has position (xyz) and normal (xyz) — 6 floats per vertex.

💡 Why 36 not 8? OpenGL draws triangles, not quads. Each face = 2 triangles. We can't share vertices across faces because each face has a different normal direction — so we duplicate all 4 corners per face rather than sharing them.

3 Write vertex shader — MVP pipeline ›
Vertex shader — full MVP, pass normal and position for later lighting
const char* vertSrc = R"(
#version 330 core
layout(location = 0) in vec3 aPos;
layout(location = 1) in vec3 aNormal;

out vec3 FragPos;   // world position (for Lab 2 lighting)
out vec3 Normal;    // world normal  (for Lab 2 lighting)

uniform mat4 uModel;
uniform mat4 uView;
uniform mat4 uProj;

void main() {
    FragPos     = vec3(uModel * vec4(aPos, 1.0));
    Normal      = mat3(transpose(inverse(uModel))) * aNormal;
    gl_Position = uProj * uView * vec4(FragPos, 1.0);
}
)";
Why pass FragPos and Normal even in Lab 1? You'll add lighting in Lab 2 without changing the vertex shader. These outputs are ready — the fragment shader in Lab 1 just won't use them yet.

Lab 2 Day 2 · Module 5 · Builds on Lab 1

Add Phong Lighting to the Vessel

Take your rotating cube from Lab 1 and add full Phong lighting — ambient, diffuse, and specular. The vessel must look three-dimensional and solid, with the light source positioned to simulate a scene light (operational room overhead or sunlight depending on context).

📋 Operational Requirements
  • →Keep Lab 1 geometry and MVP setup exactly as-is (no changes to vertex data)
  • →Light source is fixed at position (3, 3, 3) — upper-front-right corner
  • →Ambient strength: 0.1 (dark surfaces are dim but not black)
  • →Diffuse: standard Phong dot product — surface angle determines brightness
  • →Specular: Blinn-Phong with shininess 64 — moderate metallic sheen
  • →Vessel colour: navy grey (vec3(0.5, 0.55, 0.6))
1 Copy Lab 1 project and update the fragment shader only ›

Copy your entire Day2_Lab1 folder to Day2_Lab2. The vertex shader stays identical. Only the fragment shader and the uniform-sending code in the render loop change.

Fragment shader — full Phong lighting
const char* fragSrc = R"(
#version 330 core
in vec3 FragPos;
in vec3 Normal;
out vec4 FragColor;

uniform vec3  uLightPos;      // fixed at (3,3,3) — upper front right
uniform vec3  uViewPos;       // camera position for specular
uniform vec3  uObjectColor;   // navy grey: (0.5, 0.55, 0.6)
uniform vec3  uLightColor;    // white: (1, 1, 1)
uniform float uAmbientStr;    // 0.1
uniform float uShininess;     // 64.0

void main() {
    // AMBIENT — base visibility even on dark side
    vec3 ambient = uAmbientStr * uLightColor;

    // DIFFUSE — angle of surface to light
    vec3 norm     = normalize(Normal);
    vec3 lightDir = normalize(uLightPos - FragPos);
    float diff    = max(dot(norm, lightDir), 0.0);
    vec3 diffuse  = diff * uLightColor;

    // SPECULAR — Blinn-Phong half vector
    vec3 viewDir  = normalize(uViewPos - FragPos);
    vec3 halfDir  = normalize(lightDir + viewDir);
    float spec    = pow(max(dot(norm, halfDir), 0.0), uShininess);
    vec3 specular = 0.5 * spec * uLightColor;

    vec3 result = (ambient + diffuse + specular) * uObjectColor;
    FragColor = vec4(result, 1.0);
}
)";
2 Send lighting uniforms in render loop ›
Lighting uniform setup — add before render loop
// Set these once — they don't change per frame
glUseProgram(shaderProgram);
glm::vec3 lightPos(3.0f, 3.0f, 3.0f);
glm::vec3 camPos(2.0f, 1.5f, 3.0f);   // same as your view matrix eye position

glUniform3f(glGetUniformLocation(shaderProgram, "uLightPos"),    lightPos.x, lightPos.y, lightPos.z);
glUniform3f(glGetUniformLocation(shaderProgram, "uViewPos"),     camPos.x,   camPos.y,   camPos.z);
glUniform3f(glGetUniformLocation(shaderProgram, "uObjectColor"),  0.5f, 0.55f, 0.6f);  // navy grey
glUniform3f(glGetUniformLocation(shaderProgram, "uLightColor"),   1.0f, 1.0f,  1.0f);
glUniform1f(glGetUniformLocation(shaderProgram, "uAmbientStr"),  0.1f);
glUniform1f(glGetUniformLocation(shaderProgram, "uShininess"),   64.0f);
💡 uViewPos must match your lookAt eye position. If they differ, the specular highlight will appear in the wrong location relative to your camera.

Add a texture to your lit vessel. The surface texture simulates hull plating or identification markings. Combine the Phong lighting from Lab 2 with texture sampling — the final colour is lighting result × texture colour.

📋 Operational Requirements
  • →Keep Lab 2 lighting setup exactly as-is

2 Load texture and update both shaders ›
Texture load + updated vertex + fragment shaders
// ── VERTEX SHADER (add UV passthrough) ──
const char* vertSrc = R"(
#version 330 core
layout(location = 0) in vec3 aPos;
layout(location = 1) in vec3 aNormal;
layout(location = 2) in vec2 aTexCoord;   // NEW

out vec3 FragPos;
out vec3 Normal;
out vec2 vTexCoord;    // NEW — pass UV to fragment shader

uniform mat4 uModel, uView, uProj;

void main() {
    FragPos    = vec3(uModel * vec4(aPos, 1.0));
    Normal     = mat3(transpose(inverse(uModel))) * aNormal;
    vTexCoord  = aTexCoord;
    gl_Position = uProj * uView * vec4(FragPos, 1.0);
}
)";

// ── FRAGMENT SHADER (multiply lighting by texture) ──
const char* fragSrc = R"(
#version 330 core
in vec3 FragPos;
in vec3 Normal;
in vec2 vTexCoord;
out vec4 FragColor;

uniform sampler2D uTexture;
uniform vec3 uLightPos, uViewPos, uLightColor;
uniform float uAmbientStr, uShininess;

void main() {
    vec3 texColor = texture(uTexture, vTexCoord).rgb;

    vec3 ambient  = uAmbientStr * uLightColor;
    vec3 norm     = normalize(Normal);
    vec3 lightDir = normalize(uLightPos - FragPos);
    float diff    = max(dot(norm, lightDir), 0.0);
    vec3 diffuse  = diff * uLightColor;
    vec3 viewDir  = normalize(uViewPos - FragPos);
    vec3 halfDir  = normalize(lightDir + viewDir);
    float spec    = pow(max(dot(norm, halfDir), 0.0), uShininess);
    vec3 specular = 0.5 * spec * uLightColor;

    // KEY LINE: multiply ALL lighting by the texture colour
    // Phong tells us how much light — texture tells us surface colour
    vec3 result = (ambient + diffuse + specular) * texColor;
    FragColor = vec4(result, 1.0);
}
)";

// ── LOAD TEXTURE (same as D2-08 demo) ──
stbi_set_flip_vertically_on_load(true);
unsigned int tex; glGenTextures(1, &tex); glBindTexture(GL_TEXTURE_2D, tex);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_S, GL_REPEAT);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_T, GL_REPEAT);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MIN_FILTER, GL_LINEAR_MIPMAP_LINEAR);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MAG_FILTER, GL_LINEAR);
int w,h,ch; unsigned char* data = stbi_load("metal_plate.jpg", &w, &h, &ch, 0);
if(data){ glTexImage2D(GL_TEXTURE_2D,0,GL_RGB,w,h,0,GL_RGB,GL_UNSIGNED_BYTE,data);
           glGenerateMipmap(GL_TEXTURE_2D); }
stbi_image_free(data);
