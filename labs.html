Tools

Pre-Installed on Your VM

⚙️
Visual Studio 2022
C++ compiler, debugger, CMake integration. Use Developer Command Prompt for builds.
📐
CMake 3.28
Build system generator. Reads CMakeLists.txt, creates VS project files.
🪟
GLFW 3.3
Window creation, input, and OpenGL context. Used in all OpenGL labs.
🔌
GLEW 2.2
Loads modern OpenGL functions from GPU driver. Required for GL 3.3+ on Windows.
📐
GLM
Math library — vec3, mat4, perspective(), lookAt(). Matches GLSL types exactly.
🖼️
STB Image
Single-header image loader. Reads JPG/PNG for texture labs on Day 2.
🌋
Vulkan SDK 1.3
Headers, glslc compiler, validation layers, vkcube demo. Used Day 4–5.
🔍
RenderDoc
GPU frame debugger. Inspect every draw call, texture, shader. Day 3–5.
📦
Assimp
3D model loader (.obj, .fbx, .gltf). Used in Day 5 model loading lab.
💻
VS Code
GLSL syntax highlighting, Copilot integration, CMake tools extension.
